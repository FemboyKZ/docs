## Admin Commands

::: tip
Usage of commands:

In chat without with `!` or `/` in front of command.

In console with the `sm_` prefix.

You can bind keys to all the commands. Example: "bind key sm_cp"
:::

::: info
Parameters in `<>` are required.

Parameters in `[]` are optional.
:::
