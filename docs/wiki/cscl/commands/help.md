# Base Commands

::: tip
Usage of commands:

In chat with `!` or `/` in front of the command. Example: "!rtv"

In console with the `sm_` prefix. Example: "sm_rtv"

You can bind keys to all the commands. Example: "bind key sm_rtv"
:::
