# Admin Commands

::: tip
Usage of commands:

In chat with `!` or `/` in front of the command. Example: "!admin"

In console with the `sm_` prefix. Example: "sm_admin"

You can bind keys to all the commands. Example: "bind key sm_admin"
:::
