# CS:GO Commands

::: tip
Usage of commands:

In chat with `!` or `/` in front of command. Example: "!cp"

In console with the `sm_` prefix. Example: "sm_cp"

You can bind keys to all the commands. Example: "bind key sm_cp"
:::
